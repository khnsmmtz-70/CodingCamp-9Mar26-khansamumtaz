# Browser Dashboard

A minimal, customizable browser dashboard with focus timer, to-do list, and quick links.

## Features

- 🌓 **Light/Dark Mode** - Toggle between themes
- 👋 **Personalized Greeting** - Custom name with time-based greeting
- ⏱️ **Focus Timer** - Customizable Pomodoro timer (1-60 minutes)
- ✅ **To-Do List** - Add, edit, mark done, and delete tasks
- 🔗 **Quick Links** - Save favorite websites for quick access
- 💾 **Local Storage** - All data persists in your browser

## Demo

🔗 [Live Demo](https://your-username.github.io/your-repo-name/)

## Usage

Simply open `index.html` in your browser or visit the live demo link above.

### Features Guide

- **Theme Toggle**: Click the moon/sun icon in the top-right corner
- **Custom Name**: Type your name in the greeting section
- **Timer**: Set custom duration, then use Start/Stop/Reset buttons
- **Tasks**: 
  - Single-click to mark as done
  - Double-click to edit
  - Click Delete to remove
- **Links**: Add name and URL, click to open in new tab

## Tech Stack

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- Local Storage API
